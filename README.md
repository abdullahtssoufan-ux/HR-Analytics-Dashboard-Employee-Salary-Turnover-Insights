# HR Analytics Dashboard: Employee Salary & Turnover Insights

## Project Overview
This project analyzes an employee dataset to identify workforce patterns related to headcount, salary distribution, and employee turnover. The goal is to extract clear HR insights that can support better staffing, retention, and compensation decisions.

## Business Questions
- How many employees are in the dataset?
- Which department has the highest average salary?
- Which departments have the highest turnover?
- Which gender shows the highest turnover rate?

## Key Insights
- Total employees in the dataset: **200**
- Department with the highest average salary: **Quality Control**
- Departments with the highest turnover: **Logistics and HR**
- Highest turnover by gender: **Female employees (25.53%)**

## Why This Matters
These findings help HR and leadership teams understand where compensation is strongest and where retention risks may be higher. Turnover patterns can highlight departments that may need improved engagement strategies, workload review, or hiring process adjustments.

## Project Value
This analysis demonstrates how HR data can be transformed into actionable insights for:
- workforce planning
- retention strategy
- department-level performance review
- people analytics reporting
